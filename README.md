# Design and Analysis of Algorithms (DAA) in Python

Welcome to my Design and Analysis of Algorithms repository.

This repository contains Python implementations of fundamental algorithms taught in undergraduate Computer Science courses. Each program includes:

- Python implementation
- Algorithm explanation
- Time Complexity
- Space Complexity
- Sample Input and Output

---

## Topics Covered

### Sorting Algorithms

- Selection Sort
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

### Graph Algorithms

- Breadth First Search (BFS)
- Depth First Search (DFS)
- Prim's Algorithm
- Kruskal's Algorithm
- Dijkstra's Algorithm
- A* Search Algorithm

### Dynamic Programming

- Fibonacci
- 0/1 Knapsack
- Longest Common Subsequence
- Matrix Chain Multiplication
